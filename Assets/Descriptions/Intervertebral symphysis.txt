

INTERVERTEBRAL SYMPHYSIS


An intervertebral disc (or intervertebral fibrocartilage) lies between adjacent vertebrae in the vertebral column. 

Each disc forms a fibrocartilaginous joint (a symphysis), to allow slight movement of the vertebrae, to act as a ligament to hold the vertebrae together, and to function as a shock absorber for the spine.

SÍNFISIS INTERVERTEBRAL


Un disco intervertebral (o fibrocartílago intervertebral) se encuentra entre las vértebras adyacentes en la columna vertebral.

Cada disco forma una articulación fibrocartilaginosa (una sínfisis), para permitir un ligero movimiento de las vértebras, actuar como un ligamento para mantener unidas las vértebras y funcionar como un amortiguador para la columna vertebral.

;;;FR;;;



SYMPHYSE INTERVERTÉBRALE


Un disque intervertébral (ou fibrocartilage intervertébral) se situe entre les vertèbres adjacentes dans la colonne vertébrale. 

Chaque disque forme une articulation fibrocartilagineuse (une symphyse), pour permettre un léger mouvement des vertèbres, pour agir comme un ligament pour maintenir les vertèbres ensemble et pour fonctionner comme un amortisseur pour la colonne vertébrale. 

;;;PT;;;



SÍMFISE INTERVERTEBRAL


Um disco intervertebral (ou fibrocartilagem intervertebral) situa-se entre as vértebras adjacentes na coluna vertebral. 

Cada disco forma uma articulação fibrocartilaginosa (uma sínfise), para permitir um leve movimento das vértebras, atuar como um ligamento para manter as vértebras unidas e funcionar como um amortecedor para a coluna.
